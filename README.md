# SubTitulador Pro (subtitulador-swift)

A Vite + React + TypeScript project using shadcn-ui and Tailwind CSS.

## Development

Requirements: Node.js and npm.

```sh
npm i
npm run dev
```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
